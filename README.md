# Checklist PPRO — Simulador Industrial

Projeto desenvolvido em Python que simula um checklist PPRO (Programa de Pré-Requisitos Operacionais) utilizado em linha de produção industrial.

## Sobre o projeto

Baseado na experiência real como Operador de Máquinas na indústria alimentícia, o programa automatiza a verificação de pontos críticos de controle antes do início da operação.

## O que o programa verifica

- Temperatura de ponteamento (padrão: 340°C a 360°C)
- Temperatura de selagem (padrão: 240°C a 260°C)
- Pressão do ar comprimido (padrão: 5.5 a 6.5 bar)
- Peças críticas da máquina
- Telas de proteção do bico de envase
- Validade e lote das embalagens
- Conformidade das embalagens

## Relatório final

Ao concluir o checklist o programa gera um relatório com todos os alertas encontrados e indica se a máquina está liberada para operação.
